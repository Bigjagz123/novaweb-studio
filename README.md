# NovaWeb Studio

A responsive web development agency website built to demonstrate frontend development skills using HTML, CSS, JavaScript, and Bootstrap.

## Features
- Responsive layout for mobile and desktop
- Business-focused website structure
- Styled contact section with client-side validation
- Smooth scrolling navigation
- Deployed on Netlify

## Technologies Used
- HTML5
- CSS3
- Bootstrap 4
- JavaScript (ES6)
